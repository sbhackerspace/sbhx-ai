Python Connect Four Game
	By Ryan Bahneman


To play a game, open game.py and modify the players (human or AI) at the bottom, then run "python3 game.py"


To train the neural network for the NN minimax ai:
	-modify the values of training_data_generation.py to your liking
	-run "python3 training_data_generation.py"
	-wait... output will be stored in generated_data/
	-go to the octave_neural_net/ folder
	-run "octave learnNNWeights.m"
	-wait... output will be stored in generated_data/
	-copy the values in nn_training_data_output.txt into minimax_with_NN_ai.py
